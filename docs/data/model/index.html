Base Model

The SPASE metadata model can be expressed in many forms. One of those forms is as an XML Schema. An XML Schema is a machine readable form (see schema). Here you can find human readable documentation which includes standard model diagrams.

Current Release is 2.6.1 (HTML PDF XSD JSON), released: 2024-06-20, History of changes

Search the model dictionary.

When referencing the Base Model in publications use the DOI: https://doi.org/10.48322/E72C-5Y75

Draft version:
There is no draft version of this model.
