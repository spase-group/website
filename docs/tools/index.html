Tools

Reference implementations and tools for working with SPASE metadata and the SPASE framework.

Viewer
SPASE description viewer.

Viewer
View local files in a browser based SPASE description viewer. Drag and drop files into the page to view a formatted version of the SPASE XML files
Validator
Determines compliance with SPASE data model.

XML Validate
Test a web accessible SPASE description against a selected version of the data dictionary.
Editor
Web-based Editors

Web Editor
The web SPASE XML editor can be used to create or alter SPASE XML descriptions.
Resource Tools
Collections of tools and applications for working with resource descriptions

SPASE Resource Tools
A set of command-line applications to generate, validate, referentially check, use and organize resource descriptions written in SPASE XML.
Registry
Services to delivery SPASE metadata.

SPASE Registry Server
The SPASE Registry Server is a java application which can harvest resource descriptions expressed in SPASE XML and provide a search service for the descriptions. It can also chain to other registry servers and aggregate all results, returning all matches in a self-organized network of registry servers. It can be run as either a servlet or bean. The Registry Server includes a graphical tree based interface to the registries. The application calls the "resolver" service to populate and navigate through the tree.
