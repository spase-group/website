Services

SPASE Resource IDs are assigned by naming authorities. In a global system naming authorities must be unique. SPASE maintains the list of naming authorities used in our community. SPASE also administraters the metadata registry containing descriptions common entities.This are available on Github and through the SPASE SMWG registry. SPASE also maintains a registry for the Heliophysics Data Environment (HPDE).

Naming Authorities

SPASE maintains a list of naming authorities who use the "spase" scheme in resource IDs. A SPASE Resource ID is a URI and has the general form of 

spase://authority/path
To ensure global uniqueness of SPASE Resource IDs a naming authority must resource a unique name with the SPASE group. To become a naming authority and register your naming authority name with SPASE contact Todd King (tking@igpp.ucla.edu)

SMWG Registry

The SPASE Metadata Working Group (SMWG) administrators the metadata registry containing descriptions common entities. This includes descriptions of observatories, instruments, persons, registries and services. These descriptions are typically referenced by descriptions of digital resources such as numerical data, display data and catalogs. The SPASE descriptions are maintained in the metadata repository managed as a Github project at: https://github.com/hpde/spase-SMWG

SMWG Registry https://hpde.io/SMWG

HPDE Registry

SPASE hosts a registry service to provided access to descriptions of the resources in the Heliophysics Data Environment (HPDE). The SPASE descriptions included in the HPDE registry are from all the naming authority repositories maintain on Github at https://github.com/hpde

HPDE Registry http://hpde.io/
